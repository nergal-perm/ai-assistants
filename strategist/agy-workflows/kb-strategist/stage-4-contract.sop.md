# Составление месячного контракта

## Обзор

Этап формирования конкретного плана экспериментов и действий на месяц (или квартал). На основе ТОС-месяца и списка приоритетных проектов создается структурированный контракт с четкими результатами, бюджетом времени и метриками успеха.

## Параметры

- **тос_месяц** (required, assistant_created): ТОС‑месяц с гипотезой снятия и критериями успеха
- **список_проектов** (required, assistant_created): Список приоритетных проектов с карточками
- **предыдущий_контракт** (optional, user_input): Месячный/квартальный контракт с прошлого периода

**Ограничения для получения параметров:**

- Ты ДОЛЖЕН использовать ТОС-месяц и список приоритетных проектов из предыдущих этапов
- Ты ДОЛЖЕН спросить, есть ли прошлый контракт
- Ты ДОЛЖЕН предложить актуализацию прошлого контракта, если пользователь его предоставляет

## Шаги

### 1. Определение периода контракта

Установи временные рамки контракта совместно с пользователем.

**Ограничения:**

- Ты ДОЛЖЕН определить период: месяц или больше (квартал)
- Ты МОЖЕШЬ указать конкретную дату как итоговую точку (например, 1 число следующего месяца)

### 2. Заполнение результатов по проектам

Определи конкретные результаты для каждого приоритетного проекта.

**Ограничения:**

- Ты ДОЛЖЕН для каждого приоритетного проекта определить:
  - Результат на выбранный период в виде рабочих продуктов
  - Хотя бы 1 внешний результат для мира
- Ты ДОЛЖЕН помочь пользователю оценить нагрузку, ресурсы, риски, возможное сопротивление системы
- Ты ДОЛЖЕН адаптировать план на основе оценки реализуемости и рисков

### 3. Определение бюджета времени

Рассчитай и согласуй временные ресурсы на период.

**Ограничения:**

- Ты ДОЛЖЕН определить бюджет времени на выбранный период
- Ты ДОЛЖЕН рассчитать бюджет времени в среднем на неделю
- Ты ДОЛЖЕН применять принцип: ритм важнее объёма (лучше устойчивые 10 ч/нед, чем редкие «рывки»)

### 4. Планирование саморазвития (Проект развития себя)

Определи параметры саморазвития на период контракта.

**Ограничения:**

- Ты ДОЛЖЕН определить доминирующую роль месяца для саморазвития из: Ученик, Интеллектуал, Профессионал, Исследователь, Просветитель
- Ты ДОЛЖЕН выбрать практики (1–3) с целевой ступенью (Объяснение/Умение/Навык/Мастерство)
- Ты ДОЛЖЕН определить рабочие продукты месяца (1–3) в части саморазвития
- Ты ДОЛЖЕН составить список литературы для медленного чтения исходя из приоритетных проектов
- Ты ДОЛЖЕН идентифицировать риски и стоп‑лист
- Ты ДОЛЖЕН определить критерий успеха месяца

### 5. Формирование сводных метрик

Определи показатели для отслеживания прогресса.

**Ограничения:**

- Ты ДОЛЖЕН включить в контракт сводные метрики месяца:
  - Выполненность контракта
  - Перенос методов
  - Внешний итог
  - План/факт времени
  - Тренд самооценки
  - Вклад/репутация

### 6. Фиксация контракта в экзокортексе

Сохрани контракт для еженедельной коррекции.

**Ограничения:**

- Ты ДОЛЖЕН превратить понимание системы в конкретные шаги, эксперименты и привычки с понятными критериями успеха
- Ты ДОЛЖЕН зафиксировать контракт в формате, доступном для еженедельной коррекции

## Примечания

**Важные принципы:**

- Контракт — живой документ: корректируется итеративно по итогам недельных спринтов
- В контракте может быть указана конкретная дата как итоговая точка для привязки результата
- Ритм важнее объёма: лучше устойчивые 10 ч/нед, чем редкие «рывки»
- Контракт должен быть доступен для еженедельной коррекции в экзокортексе
