# Этап 4. Составление месячного/квартального контракта

**Входящие:**
- ТОС‑месяц; список приоритетных проектов

**Что делаем:**

1. Заполняем шаблон «Месячный контракт» (версия 1), в котором для каждого проекта определяем:
- Период (месяц или больше);
- Результат по каждому приоритетному проекту на выбранный период в виде рабочих продуктов и хотя бы 1 внешний результат для мира.
- Бюджет времени на выбранный период и отдельно в среднем на неделю.
2. Определяем по Проекту развития себя:
- доминирующая роль месяца для саморазвития из Ученика, Интеллектуала, Профессионала, Исследователя, Просветителя;
- Практики (1–3) с целевой ступенью (Объяснение/Умение/Навык/Мастерство);
- Рабочие продукты месяца (1–3) в части саморазвития;
- Список литературы для медленного чтения исходя из приоритетных проектов
- риски и стоп‑лист;
- критерий успеха месяца

**Выходящие:**
- Месячный контракт в экзокортексе, доступный для еженедельной коррекции, Сводные метрики месяца: выполненность контракта, перенос методов, внешний итог, план/факт времени, тренд самооценки, вклад/репутация.

**Важно:**

Спросить есть ли прошлый контракт и если пользователь его дает, что предложить его актуализацию.

Контракт — живой: корректируется итеративно по итогам недельных спринтов. В контакте может быть указана конкретная дата, например, 1 число следующего месяца как итоговая точка для привязки результата.

Ритм важнее объёма: лучше устойчивые 10 ч/нед, чем редкие «рывки».
